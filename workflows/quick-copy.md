# Quick Copy Workflow

A streamlined workflow for standalone copy requests that don't require full campaign treatment.

## When to Use

- Single page or section copy
- Quick turnaround needs
- No strategic research required
- Clear brief with context

## Workflow

### Step 1: Brief Analysis (5 min)

**Mayor Actions**:
1. Review request
2. Identify copy type and goals
3. Determine single skill needed
4. Create minimal convoy

### Step 2: Context Gathering (if needed)

**Polecat Actions**:
1. Review any provided materials
2. Note brand voice requirements
3. Identify target audience
4. Understand conversion goal

### Step 3: Copy Creation (Core)

**Skill**: `copywriting`

**Outputs**:
- Primary copy version
- 2-3 headline alternatives
- CTA variations
- Rationale notes

### Step 4: Optional Polish

**Skill**: `copy-editing`

**If needed for**:
- Refinement of existing drafts
- Brand voice alignment
- Error checking

### Step 5: Delivery

**Mayor Actions**:
1. Review output quality
2. Ensure alternatives provided
3. Package with rationale
4. Mark convoy complete

## Example Convoy

```
convoy-quick-001
├── mkt-001: copywriting (landing page hero)
└── mkt-002: copy-editing (polish) [optional]
```

## Deliverable Format

```markdown
# Copy Deliverable: [Page/Section Name]

## Primary Version

### Headline
[Recommended headline]

### Subheadline
[Supporting copy]

### Body
[Main copy]

### CTA
[Call to action]

## Alternatives

### Headline Options
1. [Option 1] - [Rationale]
2. [Option 2] - [Rationale]
3. [Option 3] - [Rationale]

### CTA Options
1. [Option 1]
2. [Option 2]

## Strategic Notes
[Why these choices were made]
```

## Timeline

| Step | Duration |
|------|----------|
| Brief analysis | 5 min |
| Context (if needed) | 10 min |
| Copy creation | 20-30 min |
| Polish (optional) | 10 min |
| **Total** | **35-55 min** |
