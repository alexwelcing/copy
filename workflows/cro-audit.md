# CRO Audit Workflow

A systematic workflow for comprehensive conversion rate optimization audits.

## When to Use

- Existing pages need optimization
- Conversion rates below benchmark
- Pre-redesign analysis
- Competitive positioning review

## Workflow

### Phase 1: Page Analysis

**Mayor Actions**:
1. Gather page URL(s) and context
2. Understand current metrics
3. Identify conversion goals
4. Create audit convoy

**Skills**:
- `page-cro` - Core analysis

### Phase 2: Deep Dive Audits

**Parallel Tracks**:

#### Track A: Conversion Elements
- `form-cro` - Form analysis
- `popup-cro` - Popup effectiveness
- `signup-flow-cro` - Registration flow

#### Track B: Supporting Elements
- `copywriting` - Copy effectiveness
- `marketing-psychology` - Persuasion audit

### Phase 3: Technical Analysis

**Skills**:
- `seo-audit` - Technical SEO
- `schema-markup` - Structured data
- `analytics-tracking` - Measurement audit

### Phase 4: Synthesis & Recommendations

**Mayor Actions**:
1. Compile all audit findings
2. Prioritize by impact
3. Create test roadmap
4. Package recommendations

## Example Convoy

```
convoy-cro-audit-001
├── mkt-001: page-cro (main landing page)
├── mkt-002: page-cro (pricing page)
├── mkt-003: form-cro (signup form)
├── mkt-004: signup-flow-cro (registration)
├── mkt-005: copywriting (copy audit)
├── mkt-006: seo-audit (technical)
├── mkt-007: ab-test-setup (test plan) [depends: all above]
```
