# SEO Campaign Workflow

A comprehensive workflow for SEO-focused marketing initiatives.

## When to Use

- New site SEO setup
- SEO improvement projects
- Programmatic SEO initiatives
- Content-driven SEO campaigns

## Workflow

### Phase 1: Audit & Research

**Skills**:
- `seo-audit` - Technical and content audit
- `competitor-alternatives` - Competitive SEO analysis

**Outputs**:
- Technical SEO audit
- Content gap analysis
- Keyword opportunities
- Competitor insights

### Phase 2: Strategy Development

**Mayor Actions**:
1. Review audit findings
2. Prioritize opportunities
3. Define content plan
4. Set up implementation convoy

**Skills**:
- `programmatic-seo` - Scalable content strategy
- `schema-markup` - Structured data plan

### Phase 3: Content Creation

**Skills**:
- `copywriting` - SEO-optimized content
- `page-cro` - User experience optimization

**Outputs**:
- Optimized page content
- Meta titles/descriptions
- Header structure
- Internal linking plan

### Phase 4: Technical Implementation

**Skills**:
- `schema-markup` - JSON-LD implementation
- `analytics-tracking` - SEO tracking setup

**Outputs**:
- Schema markup code
- Sitemap updates
- Technical specifications

### Phase 5: Measurement Setup

**Skills**:
- `analytics-tracking` - Search Console integration
- `ab-test-setup` - SEO testing framework

## Example Convoy

```
convoy-seo-001
├── mkt-001: seo-audit (comprehensive audit)
├── mkt-002: competitor-alternatives (competitive analysis)
├── mkt-003: programmatic-seo (strategy) [depends: 001, 002]
├── mkt-004: copywriting (pillar content) [depends: 003]
├── mkt-005: copywriting (supporting content) [depends: 003]
├── mkt-006: schema-markup (implementation) [depends: 004, 005]
├── mkt-007: page-cro (optimization) [depends: 004, 005]
└── mkt-008: analytics-tracking (measurement) [depends: all]
```

## Deliverable Structure

### SEO Audit Report
- Technical issues (critical, high, medium, low)
- Content opportunities
- Competitive gaps
- Quick wins

### Keyword Strategy
- Target keywords by page
- Search intent mapping
- Content recommendations
- Priority ranking

### Content Briefs
- Target keyword
- Search intent
- Competitor analysis
- Outline/structure
- Internal linking

### Technical Specifications
- Schema markup code
- Meta tag templates
- URL structure
- Sitemap updates

### Implementation Checklist
- [ ] Technical fixes
- [ ] On-page optimization
- [ ] Schema implementation
- [ ] Content updates
- [ ] Internal linking
- [ ] Measurement setup
