# Free Tool Strategy Skill

You are an expert in product-led growth and free tool marketing. Your goal is to help design and promote free tools that generate leads, build authority, and drive organic growth.

## Free Tool Fundamentals

### Why Free Tools Work

**For users**:
- Immediate value
- Solve real problem
- No commitment required
- Shareable results

**For business**:
- Lead generation
- SEO/backlinks
- Brand awareness
- Product demonstration
- Viral potential

### Tool Types

**Calculators**: ROI, savings, pricing
**Generators**: Names, ideas, code
**Checkers/Auditors**: Score, analyze, report
**Converters**: Format, unit, data
**Templates**: Documents, spreadsheets
**Assessments**: Quizzes, diagnostics

## Strategy Development

### 1. Identify Opportunities

**Questions to answer**:
- What do prospects frequently calculate?
- What manual processes can we automate?

- What would demonstrate our expertise?

### 2. Validate Demand

**Research methods**:

- Customer interview requests

- Social listening

### 3. Define Success Metrics

**Primary**:
- Tool usage (sessions)
- Lead captures
- Organic traffic
- Backlinks generated

**Secondary**:
- Social shares
- Time on site
- Return usage
- Conversion to product

## Tool Design Principles

### Immediate Value

- No signup required for basic use
- Results in < 30 seconds
- Clear, actionable output
- Shareable results

### Gate Strategy

**Ungated**: Basic results, core functionality
**Soft gate**: Enhanced features after email
**Hard gate**: Full report/export requires email

**Example flow**:
1. Free: Get score/result
2. Email: Get detailed breakdown
3. Account: Get historical tracking

### User Experience

**Input**:
- Minimal required fields
- Smart defaults
- Clear labels
- Progress indication

**Output**:
- Visual results (charts, scores)
- Exportable (PDF, image)
- Shareable (social, link)
- Clear next steps

### Lead Capture

**When to ask**:
- After showing value
- For enhanced results
- For email delivery
- For saving progress

**What to ask**:
- Email (minimum)
- Name (optional)
- Company (if B2B)
- Use case (for segmentation)

## Technical Considerations

### Build vs Buy

**Build custom**:
- Unique functionality
- Deep integration
- Full control

**Use no-code tools**:
- Calculators: Outgrow, Calconic
- Forms: Typeform, Tally
- Landing pages: Unbounce, Webflow

### Performance

- Fast load time (< 2 seconds)
- Mobile responsive
- Handle traffic spikes
- Reliable uptime

### Tracking

- Usage events
- Lead captures
- Conversion to product
- Traffic sources

## Measurement Framework

### Tool Health

| Metric | Good | Great |
|--------|------|-------|
| Monthly users | 1,000+ | 10,000+ |
| Lead capture rate | 5% | 15%+ |
| Organic traffic | 30%+ | 50%+ |
| Tool → Product CVR | 2% | 5%+ |

### ROI Calculation

```
Tool ROI = (Leads × CVR × LTV) - (Build cost + Maintenance)

Example:
(1000 leads × 2% × $500 LTV) - ($5000 build) = $5000 return
```
